# HUANANZHI-B85-ITX-Hackintosh | 华南B85-ITX 黑苹果

## TECH LIST | 配置清单
- CPU: i5-4690
- MEM: DUAL-CHANNEL 16G=8x2
- MOTHER-BOARD: HUANANZHI B85 ITX
- NVME: WD m2

## CHECK POINT | 黑苹果功能点核查
- DISPLAY: IGPU 👌
- AUDIO: ALC662 workwith appalc alcid=17 👌
- NETWORK: RTL8111 👌
